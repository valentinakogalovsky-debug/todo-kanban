# ToDo Kanban

Приложение-доска для управления задачами в стиле Kanban.

## Идея проекта

Пользователь сможет создавать задачи, распределять их по колонкам и отслеживать прогресс работы.

## Планируемые возможности

- Колонки задач: To Do, In Progress, Done
- Добавление и удаление задач
- Редактирование текста задачи
- Перемещение задач между колонками
- Сохранение задач в браузере
